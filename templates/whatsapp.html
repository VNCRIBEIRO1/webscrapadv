Configuração do WPPConnect

Como configurar o WPPConnect Server:

  1. Instale o Node.js (v16+)
  2. Clone: git clone https://github.com/wppconnect-team/wppconnect-server.git
  3. Instale: cd wppconnect-server && npm install
  4. Configure o .env com a SECRET_KEY
  5. Inicie: npm start (padrão: porta 21465)
  6. Escaneie o QR Code com o WhatsApp

O WPPConnect é um projeto open-source. Use de forma responsável e respeite os termos de uso do WhatsApp.
